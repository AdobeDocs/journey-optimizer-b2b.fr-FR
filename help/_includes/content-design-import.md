# Création de contenu - import HTML

Le contenu importé peut être :

* Fichier HTML avec feuille de style incorporée
* Un fichier .zip qui comprend un fichier HTML, la feuille de style (.css) et les images

  >[!NOTE]
  >
  >Il n’existe aucune contrainte sur la structure des fichiers .zip. Cependant, les références doivent être relatives et s’ajuster à l’arborescence du dossier .zip. Les images sont toujours chargées vers le référentiel de ressources Marketo Engage connecté. Si vous souhaitez gérer les fichiers image dans Experience Manager Assets, chargez-les séparément dans cette interface d’application et modifiez les liens d’image dans l’espace de conception d’e-mail.

_Pour importer un fichier contenant du contenu HTML :_

1. Sur la page d’accueil de la conception, sélectionnez l’option **[!UICONTROL Importer HTML]**.

1. Faites glisser et déposez le fichier HTML ou .zip contenant le contenu HTML, puis cliquez sur **[!UICONTROL Importer]**.
